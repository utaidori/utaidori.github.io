Windows11 でのトラブルと解決

トラブルをどう解決したかを記録しました。

Windows11 OS

アプリ起動のショートカットキーを設定した元のファイルが見つからない

スタートメニュー内のショートカットファイルだった。
エクスプローラーで開く: %AppData%\Microsoft\Windows\Start Menu\Programs

Github Desktop

Pull しないままファイルを編集し Commit したため Push が失敗する

アプリ左下に Undo ボタンがある。コミットを取り消し、各ファイルを右クリックして Discard changes を選ぶ。変更箇所は消えるので、先に当該箇所をコピーして Pull の後に改めて編集する。

Vim (gVim 9.0)

:terminal で起動した PowerShell でのトラブル

[Ctrl] + [Backspace] で単語削除できない

<C-BS> を押下すると [8;5u という文字列が表示されてしまう。

tmap で下記のようにリマップする。
:tnoremap <C-BS> <C-w>.

タブを移動できない

<C-Tab> に gt （次のタブに移動）を割り当てたが機能しない。

tmap で下記のようにリマップする。
:tnoremap <C-Tab> <C-w>gt
または
:tnoremap <C-Tab> <cmd>tabn<cr>

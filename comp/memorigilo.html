Windows11 環境下でのトラブルと解決方法の覚書

Windows11 OS

アプリを起動するショートカットキー。設定元のファイルが見つからない

問題：

ショートカットファイルのプロパティの「ショートカット」タブ、「ショートカットキー(K)」の項目で、アプリを起動するショートカットキーを設定した。しかし元のファイルがデスクトップにも Program Files にもない。

解決策：

スタートメニュー内のショートカットファイルである可能性が高い。
エクスプローラーで開く: %AppData%\Microsoft\Windows\Start Menu\Programs

Github Desktop

Pull せずにファイルを編集し Commit したため Push が失敗する

問題：

Pull する前にファイルを編集し Commit してしまい、コンフリクトが発生した。

解決策：

アプリ左下に Undo ボタンがある。コミットを取り消し、各ファイルを右クリックして Discard changes を選ぶ。すでに重要な変更を行っている場合は、変更部分をコピーして Pull の後に改めて行う。（原始的だが迷いの少ない方法。）

Vim (gVim 9.0)

:terminal で起動した PowerShell

[Ctrl] + [Backspace] で単語削除

問題：

:terminal で起動した PowerShell で <C-BS> を押下すると [8;5u という文字列が表示されてしまう。

解決策：

tmap で下記のようにリマップする。
:tnoremap <C-BS> <C-w>.

タブを移動する

問題：

<C-Tab> に gt （次のタブに移動）を割り当てていたが、 :terminal で起動した PowerShell で <C-Tab> でタブの移動ができない。

解決策：

tmap で下記のようにリマップする。
:tnoremap <C-Tab> <C-w>gt
:tnoremap <C-S-Tab> <C-w>gT
または
:tnoremap <C-Tab> <cmd>tabn<cr>
:tnoremap <C-S-Tab> <cmd>tabp<cr>

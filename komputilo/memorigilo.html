Windows11 で遭遇したトラブルとその解決方法（覚え書き）

Windows11 OS

アプリ起動のショートカットキーを設定した元のファイルが見つからない

スタートメニューフォルダ（以下に書いたフォルダ）内のショートカットファイルだった。
%AppData%\Microsoft\Windows\Start Menu\Programs

Github Desktop

Pull しないままファイルを編集し Commit したため Push が失敗する

アプリ左下の Undo ボタンを押してコミットを取り消す。
各ファイルを右クリックして Discard changes を選ぶ。
Discard changes を実行すると、今回行った変更が消える。

変更を維持したい場合は次の手順で行う。

  1. 各ファイルの今回変更した箇所をコピーしておく。
  2. Undo ボタンでコミットを取り消す。
  3. 各ファイルを右クリックして Discard changes を選択。
  4. Pull を行う。
  5. コピーしておいた箇所をペーストする（あるいは置換する）。
  6. コミットを行う。

Vim (gVim 9.0)

:terminal で起動した PowerShell でのトラブル

[Ctrl] + [Backspace] で単語削除できない

<C-BS> を押下すると [8;5u という文字列が表示されてしまう。

tmap で下記のようにリマップする。
:tnoremap <C-BS> <C-w>.

タブを移動できない

<C-Tab> に gt （次のタブに移動）を割り当てたが機能しない。

tmap で下記のようにリマップする。
:tnoremap <C-Tab> <C-w>gt
または
:tnoremap <C-Tab> <cmd>tabn<cr>

Python で常に画面上に現在日時（日にち・曜日・時刻）を表示する

  • 概要：Windows のディスプレイ右上に時計を常時表示（表示オン・オフをホットキーで切り替える）
  • 機能
    • 現在日時、曜日、バッテリー残量を表示する
    • 表示オン・オフをホットキーで切り替える（AutoHotKeyを使う）
  • 挙動
    • .pyw ファイルとして実行。ウインドウは常に最前面に表示する（フォーカスは奪わない）
    • バッテリー残量（パーセント）と残り駆動時間を表示（残り時間は充電時には ... と表示）
    • 閉じるボタンがないので以下のような方法で終了する
      • シェルの taskkill コマンド
      • タスクマネージャーからタスクを終了
      • フォーカスが合っている状態で Alt+F4
  • AutoHotKey で表示のオン・オフを切り替える
    • AutoHotKey で時計の起動・終了をトグルするキーを設定する
    • AutoHotKey を使えば（前項のような面倒がなく）簡単に時計をトグルできる
    • ※Python のみで表示をオン・オフする方法は模索せず、手軽な方法を採用した

Python (3.11.3)

import datetime
import time
import psutil
import tkinter as tk

def get_time():
    now = datetime.datetime.now()
    day_of_the_week = ['月', '火', '水', '木', '金', '土', '日']
    label_time['text'] = f'{now.hour:02}:{now.minute:02}'
    label_date['text'] = f'{now.month:02}/{now.day:02}'
    label_day['text'] = f' {day_of_the_week[now.weekday()]} '

    # 次の更新までの残り秒数を計算
    remaining_seconds = 60 - now.second

    if remaining_seconds == 0:
        remaining_seconds = 1

    # 次の更新までの時間後にget_time()を再帰的に呼び出す
    root.after(remaining_seconds * 1000, get_time)

def get_battery_info():
    battery = psutil.sensors_battery()
    plugged = battery.power_plugged
    percent = battery.percent
    time_left = f'...'

    if not plugged:
        time_left_secs = battery.secsleft
        if time_left_secs > 0:
            time_left_mins = time_left_secs // 60
            time_left_hours = time_left_mins // 60
            time_left_mins %= 60
            time_left = f'{time_left_hours:02}h{time_left_mins:02}'

    label_battery.config(text=f'{percent}% ({time_left})')
    root.after(60000, get_battery_info)


root = tk.Tk()
root.title('Time')

label_time = tk.Label(
  root, font=("Lato", "18", "bold")
)

label_date = tk.Label(
  root, font=("Lato", "10")
)

label_day = tk.Label(
  root, font=("BDF東雲ゴシック", "9"), pady="3"
)

label_battery = tk.Label(
  root, font=("Lato", "8"), foreground="#888888"
)

label_day.configure(
  foreground="#f0f0f0",
  background="#888888",
  border=0
)

label_time.place(x=0, y=14)
label_date.place(x=0, y=0)
label_day.place(x=47, y=0)
label_battery.place(x=0, y=42)

get_time()
get_battery_info()

root.overrideredirect(True)
root.geometry("70x62-0+0")
root.attributes("-topmost", True)

root.mainloop()
  • 設定の変更
    • "BDF東雲ゴシック", "Lato" はフォント名なので任意のフォントに変更する
    • フォントの変更に伴い、レイアウトの必要に応じて root.geometry() の引数を変更する

; 左Windows + T で 起動/終了 を切り替え
<#t::
{
    if WinExist("Time")
    {
        ; ウィンドウが存在すれば終了させる
        Run 'taskkill /F /FI "WINDOWTITLE eq Time"', , "Hide"
    }
    else
    {
        ; ウィンドウが存在しなければ起動する
        Run "C:\Users\" . A_UserName . "\pyw\time.pyw"
    }
}
  • 設定の変更
    • time.pyw を置くフォルダは一例
    • A_UserName はユーザー名を格納したAHKの組み込み変数
